“Outils”

  • Éditeur de code (VS Code, Sublime...)

  • Langages :
    • HTML / CSS
    • JavaScript
  • Serveur local (XAMPP, WAMP...)
HTML
Langage de balisage utilisé pour structurer le contenu d’une page web.
CSS
Langage servant à mettre en forme et styliser les pages web.
JavaScript (JS)
Langage de programmation permettant d’ajouter de l’interactivité aux pages web.
